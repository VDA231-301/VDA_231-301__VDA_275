# VDA 275 – Formaldehyde Emission Test Results
## JSON Subschema according to VDA 231‑301
🔗 This subschema is part of the VDA 231‑301 ecosystem
→ https://github.com/VDA231-301
## Purpose and Scope

This repository provides a **specific JSON subschema for the digital exchange of test results according to VDA 275**,  
*“Determination of formaldehyde emissions from materials”*.

The subschema defines a **machine‑readable representation of formaldehyde emission test results** and is intended to support the standardized, transparent, and interoperable exchange of laboratory data along the supply chain.

It is based on the **VDA 231‑301 recommendation** and represents a specialized extension of the generic VDA 231‑301 JSON schema.

---

## Position within VDA 231‑301

Within the VDA 231‑301 ecosystem:
- the **generic schema** defines the common structure for digital test reports,
- this **VDA 275 subschema** specifies the data elements required to represent test results obtained using the VDA 275 method.

The subschema ensures that formaldehyde emission results generated by different laboratories can be exchanged and interpreted in a **consistent and system‑independent manner**.

---

## Relation to the VDA 275 Test Method

VDA 275 defines a **laboratory test method** for determining formaldehyde emissions from materials used in vehicle interiors.

⚠️ **Important clarification:**
- This repository **does not describe the test method itself**.
- It does **not define test execution, sampling, evaluation, or limit values**.
- It provides a **technical data structure only** for representing the *results* of a VDA 275 test in digital form.

The authoritative description of the VDA 275 test method is published exclusively by the VDA and is available via the **VDA Webshop**.

---

## Applicability Beyond the Automotive Industry

Although VDA 275 originates from the automotive interior context, **formaldehyde emission testing is relevant across many industries**, for example:
- consumer products
- furniture and interior materials
- construction products
- polymers, textiles, and foams

The subschema is therefore **not limited to automotive use cases**.  
It can be used wherever formaldehyde emission test results need to be:
- documented,
- exchanged between organizations,
- or integrated into digital compliance and quality workflows.
